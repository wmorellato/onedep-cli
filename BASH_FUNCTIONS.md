# OneDep Manager Bash Functions

Generated by `onedep paths generate-funcs`

## Setup

Source the generated file in your shell:
```bash
source ~/.onedep_funcs
```

Add to your `.bashrc` or `.bash_profile` for automatic loading.

## Environment Variables

| Variable | Description |
|----------|-------------|
| `ODM_TEMPDEP` | Base path for temporary deposit directories |
| `ODM_DEPOSIT` | Base path for deposit directories |
| `ODM_DEPOSIT_UI` | Base path for deposit UI directories |
| `ODM_ARCHIVE` | Base path for archive directories |
| `ODM_SESSION` | Base path for session directories |
| `ODM_UPLOAD` | Base path for upload directories |
| `ODM_PICKLES` | Base path for pickles directories |
| `ODM_WFINST` | Base path for workflow instance directories |
| `ODM_CCID` | Base path for chemical component dictionaries |
| `ODM_WFXML` | Base path for workflow XML files |

## Navigation Functions (cd)

### `cdt [identifier]`
**Arguments:** `identifier` (optional) - Deposition ID
**Purpose:** Change directory to temporary deposit location. If no identifier provided, goes to root.

## Editor Functions (vi)

### `viccid <identifier>`
**Arguments:** `identifier` (required) - Chemical component ID (e.g., "ABC")
**Purpose:** Open chemical component dictionary file in vi. Path structure: `$ODM_CCID/<first_letter>/<identifier>`

### `viwfx <identifier>`
**Arguments:** `identifier` (required) - Workflow identifier
**Purpose:** Open workflow XML file in vi. Opens `$ODM_WFXML/<identifier>.xml`

### `viwfi <dep_id> <wfinst_id>`
**Arguments:**
- `dep_id` (required) - Deposition ID
- `wfinst_id` (required) - Workflow instance ID

**Purpose:** Navigate to workflow instance directory and open in vi. Path: `$ODM_WFINST/<dep_id>/<wfinst_id>`

## Examples

```bash
# Navigate to specific deposit
cdd D_1234567890

# Navigate to deposit root directory
cdd

# List all sessions
lss

# List specific session
lss mysession123

# Edit chemical component
viccid ABC

# Edit workflow XML
viwfx DepositWorkflow

# Open workflow instance
viwfi D_1234567890 W_001
```
